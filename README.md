# 🎮 Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe** game developed using **HTML, CSS, and JavaScript**. The game features two-player gameplay, winner and draw detection, and a restart option while demonstrating DOM manipulation, event handling, and core JavaScript logic.

---

## 📌 Features

- 🎯 Two-player gameplay (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart game functionality
- 📱 Responsive and user-friendly interface
- ⚡ Fast and lightweight application
- 🌐 Runs directly in any modern web browser

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the game
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6)** – Game logic and interactivity

---

## 🎮 How to Play

1. Player **X** starts the game.
2. Players take turns placing their symbol (**X** or **O**) on the board.
3. The first player to get **three symbols in a row** (horizontal, vertical, or diagonal) wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click the **Restart** button to play again.
